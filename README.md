# 铁路环游游戏 - Railway Monopoly

一个用Python实现的传统铁路环游游戏，支持人机对战。

## 功能特性

- 🎮 **人机对战**:  与AI对手进行竞争
- 🎲 **掷骰子系统**: 随机移动机制
- 🏢 **物业购买**: 购买和出租站点赚取金钱
- 💰 **资金管理**:  跟踪玩家资金和破产情况
- 🎨 **图形界面**: 使用tkinter实现的友好UI
- 📊 **游戏信息面板**: 实时显示玩家状态

## 游戏规则

1. **开始资金**: 每个玩家开始时有2000元
2. **掷骰子**: 每回合掷两个骰子，前进相应的格子
3. **绕过起点**: 经过起点时获得200元
4. **购买物业**: 可以购买未拥有的站点
5. **支付租金**: 登陆他人物业时需要支付租金
6. **税收**: 某些格子会扣税
7. **破产规则**: 资金不足以支付时破产，游戏结束
8. **获胜**:  最后一个未破产的玩家获胜

## 安装

```bash
# 克隆仓库
git clone https://github.com/cz-2006/railway-monopoly.git
cd railway-monopoly

# 安装依赖（通常tkinter已预装）
pip install -r requirements.txt
```
